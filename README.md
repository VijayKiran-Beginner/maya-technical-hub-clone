# Maya Technical Hub Website Clone

A responsive and interactive front-page clone of **Maya Technical Hub** (maya.technicalhub.io) built using **HTML, CSS, and Bootstrap**.

### Features
- Fully responsive design (Mobile + Desktop)
- Interactive navigation menu
- Modern and clean UI matching the original website
- Smooth hover effects and animations
- Well-structured sections (Hero, Services, About, Contact, etc.)
- Built with Bootstrap for fast and responsive layout

### Technologies Used
- HTML5
- CSS3
- Bootstrap 5

### Project Goal
To replicate the look and feel of the official Maya Technical Hub website with clean code and responsive design.

### Future Enhancements
- Convert into a full multi-page website
- Add backend for contact form
- Make it dynamic using JavaScript frameworks (React/Vue)
